# 👻 GhostTrack

GhostTrack is a Python-based information gathering utility designed for learning and authorized network administration.

## Features

- 🌍 IP Information Lookup
- 📱 Phone Number Information
- 👤 Username Lookup
- 🌐 Show Your Public IP
- 🎨 Colorful Terminal Interface
- ⚡ Fast and Lightweight

## Requirements

- Python 3.9+
- Termux (Android) or Linux

## Installation

### Termux

```bash
pkg update && pkg upgrade
pkg install python git
pip install -r requirements.txt
git clone https://github.com/AsGamerz389/GhostTrack.git
cd GhostTrack
python GhostTR.py
```

### Linux

```bash
sudo apt update
sudo apt install python3 python3-pip git
git clone https://github.com/AsGamerz389/GhostTrack.git
cd GhostTrack
pip3 install -r requirements.txt
python3 GhostTR.py
```

## Dependencies

- requests
- phonenumbers

Install manually

```bash
pip install requests phonenumbers
```
